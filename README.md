# 💻 Portfólio de Automação com Python

Este repositório contém o código do meu site de portfólio, onde apresento projetos reais de automação desenvolvidos com Python.

---

## 📌 Sobre

O objetivo deste site é demonstrar, de forma prática, como automações podem eliminar tarefas manuais, reduzir erros e aumentar a produtividade.

Aqui estão incluídos exemplos reais de soluções aplicadas em cenários do dia a dia.

---

## ⚙️ O que você vai encontrar

- Web scraping (coleta automática de dados)
- Geração de planilhas Excel
- Extração de dados de PDFs
- Automação de sistemas (RPA)
- Fluxos completos de automação

---

## 🎯 Projetos apresentados

### 🔹 Scraping → Planilha
Coleta automática de dados de sites e organização em Excel.

### 🔹 Planilha → Sistema
Preenchimento automático de sistemas com dados estruturados.

### 🔹 PDF → Excel
Extração de informações de documentos PDF para planilhas.

### 🔹 Fluxo Completo
Automação ponta a ponta: coleta, processamento e inserção de dados.
